# OthelloJosekiMaster

オセロの定石の練習する為のアプリです。
オセロの序盤の打ち方である定石について、クイズ形式で最善手が打てるよう練習するアプリ。

## 機能
### 出題

虎定石やうさぎ定石といった名前が付いている定石について、最善の局面や最善から評価値が若干低い局面を出題し、その局面の最善手を選べるか出題する

- トップ画面（定石選択画面）
  - 最初の画面で定石一覧を表示し、ユーザーに選択させる。
- 出題画面
  - ユーザーが定石を選択しら出題画面に切り替え、出題を開始する。以後最初の画面に戻るまで選択した定石に対する繰り返し行う
  - 出題画面上に最初の画面に戻るボタンを用意する。
- 出題内容
  - 選択した定石の進行で、最善の進行及び最善でない手を打った局面を出題する
  - 再出題
    - 再出題時は３回同じ局面で正解したら以後出題しないようにする。

### 回答

- 回答者は、オセロ盤上で打つことが可能なマスをタップすることで回答する。
- オセロのルール上打つことができないマスをタップした場合は、「そのマスには打てません」をダイアログ表示し、有効な場所に打つよう促す。
- オセロ盤の有効手をマスをタップした場合のみ、石をひっくり返す。
- 石をひっくり返すときはアニメーションを入れる。

### 回答後の正誤照会

- 回答後、回答手が正解か不正解を表示する。
- 回答後、回答手と回答手以外の着手可能な手の評価値を全て表示する。
- 正解手の評価値は水色で表示する。
- 不正解手の評価値は赤色で表示する。
- 回答手以外の評価値が回答手と同じかそれ以上の値の場合は緑色、回答手よりも評価値が低い場合はオレンジ色で表示する。

### 回答後の出題の内容への反映

- 回答者の指し手の正解 or 不正解 の覚えておく。この情報は定石毎に分類する。

### 回答後の画面状態

- 回答後の画面内で、次の出題をするか、トップ画面に戻るかボタンで選べるようにする。

## 実装方法

### 開発環境

- 対象プラットホームはAndroidとする。

### 画面

Jetpack Composeで実装する。

### オセロのWebAPI

Edax API を使用して指し手を評価する。
