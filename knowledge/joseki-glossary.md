# オセロ用語辞典

オセロ・定石に関する専門用語をまとめる。
spec や設計ドキュメントで用語を使う際の共通定義として参照する。

**注**: **【要確認】** タグの箇所はユーザー (オセロ経験者) のレビューで正誤を確認してほしい。

---

## ゲーム段階

### 序盤 (Opening)
最初の1手から概ね 20手前後までの段階。定石 (joseki) が存在し、研究が進んでいる。

---

## 盤面の位置

### 隅 (Corner)
盤の 4 隅にあたるマス: `a1`, `a8`, `h1`, `h8`。隅に置いた石は絶対にひっくり返されない (確定石) ため、最も価値が高い。

### X 打ち / Xマス (X-square)
隅の斜め1マス隣: `b2`, `b7`, `g2`, `g7`。
ここに石を置くと相手に対角の隅を取られやすくなるため、序盤・中盤では原則として避ける **【要確認: 座標】**。

### C 打ち / Cマス (C-square)
隅の直線 1 マス隣: `a2`, `b1`, `a7`, `b8`, `h2`, `g1`, `h7`, `g8`。
X マスほどではないが同様に危険 **【要確認: 座標・定義】**。

### 辺 (Edge)
盤の外周 (1行目・8行目・a列・h列)。辺の石の安定性は隣接する空きマスや隅の状況で変わる。

---

## 石の安定性

### 確定石 (Stable Disc)
一度置かれた後、ゲーム終了まで **絶対にひっくり返らない** 石。以下の条件がすべて満たされると確定石になる **【要確認: 条件の正確な定義】**:

1. 隅に接している、または
2. 隅を起点として辺が連続して埋まっており、かつ行・列方向に別の確定石に隣接している

確定石を多く持つほど終盤で有利。

### 不安定石 (Unstable Disc)
将来ひっくり返される可能性がある石。特に中央付近は序盤に不安定になりやすい。

---

## 合法手と手番

### 合法手 (Legal Move)
オセロのルールで打てる手 (`othello-rules.md` 参照)。

### 手番 (Tempo / Turn)
着手できる権利。「手番を持つ」= 合法手の選択肢が多い = 有利。

### 合法手数 (Mobility)
ある局面で自分が打てる合法手の数。合法手数が多いほど戦略の幅が広い。
相手の合法手数を減らして選択肢を奪う戦略を「mobility 戦略」という **【要確認: 用語】**。

### 強制パス (Forced Pass)
合法手がなく、やむなくパスになること。対戦相手を強制パスに追い込むと手番を得られる。

---

## 理論・戦略

### 開放度理論 (Liberty Theory)
着手した石の周囲の空きマス数の合計を最小化することで、相手の合法手の増加を抑えるという考え方 **【要確認: 「開放度」の定義の揺れ】**。

- **開放度 (Openness)**: ある着手によって新たに生じる「隣接空きマス数の増加量」を指すことが多い
- 開放度が低い着手 = 相手の選択肢を増やしにくい = 良い手とされる
- 序盤〜中盤で特に重視される

---

## 定石

### 定石 (Joseki)
序盤における **最善とされる着手手順** の体系。双方が最善を尽くした場合の「公認の進行」。
定石から外れると相手に有利な変化を与える可能性がある。

本アプリの出題対象。虎定石・うさぎ定石・牛定石などの名称が付いているものを扱う。

### 変化 (Variation)
定石の途中から枝分かれする進行。定石本線以外の着手に対する対応手順も含まれる。

### 最善手 (Best Move)
ある局面で評価値が最も高い (または最も損をしない) 着手。Edax 等のエンジンで算出する。

---

## 評価・エンジン

### 評価値 (Evaluation / Disc Count Difference)
ある局面の有利不利を数値化したもの。Edax では **終局時の石数差** (黒 - 白) として表される。

- 正の値: 黒有利
- 負の値: 白有利
- 0: 互角

本アプリでは Edax API が返す評価値を「正解」「不正解」の判定と解説表示に使用する。

### Edax
オセロの強力なオープンソースエンジン。深い読み (完全読みや深い確率読み) により正確な評価値を算出できる。本アプリで評価値を取得する際に使用する。API の詳細 (エンドポイント・認証) は実装フェーズで確定する。

## 本アプリで使う用語の対応

| 本アプリの UI 表現 | 対応する用語 |
|---|---|
| 最善手 | Best Move (評価値が最高の着手) |
| 評価値 | Edax が返す disc count difference |
| 正解 | 着手が最善手と一致 (または評価値差が閾値内) |
| 不正解 | 着手が最善手と不一致 (評価値が低下) |
| 定石の局面 | joseki position |
| 再出題 | 不正解だった局面を再度出題する機能 (正解 3 回で対象から除外) |
